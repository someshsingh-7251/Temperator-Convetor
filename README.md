    <h1>Temperature Converter</h1>
    <p>Welcome to the <strong>Temperature Converter</strong> project! This tool allows you to easily convert temperatures between Celsius, Fahrenheit, and Kelvin using a simple user interface.</p>
    <h2>Features</h2>
    <ul>
        <li>Convert from Celsius to Fahrenheit and Kelvin.</li>
        <li>Convert from Fahrenheit to Celsius and Kelvin.</li>
        <li>Convert from Kelvin to Celsius and Fahrenheit.</li>
        <li>User-friendly interface for easy temperature conversion.</li>
    </ul>

  <h2>Technologies Used</h2>
    <ul>
        <li>HTML5</li>
        <li>CSS3</li>
        <li>JavaScript</li>
    </ul>

  <h2>How to Use</h2>
    <ol>
        <li>Clone the repository to your local machine:</li>
        <pre><code>git clone https://github.com/someshsingh-7251/temperature-converter.git</code></pre>
        <li>Open the <code>index.html</code> file in your browser.</li>
        <li>Enter a temperature value in the input field.</li>
        <li>Select the temperature scale you want to convert from and to.</li>
        <li>Click the "Convert" button to see the result.</li>
    </ol>
